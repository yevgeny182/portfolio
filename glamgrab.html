A Mobile Development project made in Android Native and SQLite

Information & Purpose

Java is the native language used by Android, applications that communicate with the operating system and directly use the hardware uses it. This language allows the creation of any program and supports almost all types of machines, and OS X be it Android, Windows, or Linux ( Gill, 2024 )

An e-commerce system is a digital platform facilitating the buying of goods and or services on the internet. It provides a convenient way in the shopping experience for the consumers, allowing them to browse compare and purchase products on their fingertips.

Landing Screen (Home)

The Landing Screen is the default home page of the app allowing users to browse and check product listings

On this page users can browse product listings and select by category or simply search the product on the search text bar.

Discover Screen

Discover Screen shows the categories displayed in a card layout, tapping on "Shop Now" will route user to all products relating to the category.

E.g., tapping on "Shop Now" in the women card layout will display all products in the women's category.

Likes Screen

The crew can easily add, update, or delete new or existing products on this page, streamlining operations by eliminating manual processes and making management more efficient.

Account Screen

The crew can easily add, update, or delete new or existing products on this page, streamlining operations by eliminating manual processes and making management more efficient.

Access

This mobile app is a course project in Mobile Development, currently available on GitHub. While it includes around 80% of the planned UI and 40% of the backend functionality, it has the potential to evolve into a fully functional app.
