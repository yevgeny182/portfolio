Time Management System

MySQL PHP Laravel JQuery

Purpose

A basic attendance tracker system designed for tracking time logs, monitoring attendance and assigning work hours efficiently.

An admin can log the user in or out on their behalf, assign hours to a specific user and can add or remove them as well.

Add User Feature

The add user feature, allows the admin to add a new record into the system.

Once added, their record will appear on the table with the action to login, logout, display the time logged and assign hours

Assign Hours Feature

The Assign Hours allows the admin to assign hours to a specific employee / user, in which if they time in then time out according to their time logged their assigned remaining time will be reduced.

Viewing User Record as Logged in

For monitoring purposes, the admin can view employee records and their timestamps, and they have the option to remove the user.

Incomplete Feature: Import Users

This feature is supposedly going to import multiple user records from a spreadsheet. This feature cannot be completed due to library problems with excel/spreadsheet conflicting with Laravel which I plan to put a fix soon.
