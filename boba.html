A Web Development Project Made using PHP Vanilla and MySQL

Purpose

A potential website for a milk tea shop designed to handle orders and manage products and categories, addressing essential day-to-day business needs. With the rising trend of milk tea shops in 2021, this system effectively alleviates manual processes, streamlining operations and enhancing efficiency.

Order Page

The Order Page, also known as the dashboard of the website allows the crew to create orders.

To create an order, the crew can simply input the customer name, select an item by clicking the checkbox, specify the quantity, and then save the information. This streamlined process ensures efficient order management.

Sales Page

The Sales Page presents a record of each order created from the Order Page. A table is shown displaying the date, customer name, and the order price, labeled as "Total Sales."

The "Details" section provides the full order breakdown, which can be viewed in a modal by clicking the "View" button.

Product Page

The crew can easily add, update, or delete new or existing products on this page, streamlining operations by eliminating manual processes and making management more efficient.
