Geolocation Rent Finder

A prototype for a rent finder app

Purpose

Geolocation Rent Finder is a prototype designed to help users find rental spaces according to a person's salary or in a specific area. After search results are generated, users can filter options by price, type (dorm or home), gender preference, pet allowance, and number of beds (Similar experience to browsing in an e-commerce app).

The prototype supports two user roles: renters and homeowners. Homeowners can post listings directly through the app, and renters can browse and inquire spaces.

A unique feature of this prototype is its occupancy management tool for homeowners, which enables real-time tracking of availability for each listing. Renters can see general occupancy (e.g 50/95 rented rooms), homeowners have a more detailed view of data for each renter.

Renter

On screen are the Figma frames for the Renter's user flow, which includes key features like authentication, messaging, and the core functionality of finding rental spaces through search.

The design theme for the Renter's UI is sage green this is to signify what user type they are and also for its aesthetic appeal.
